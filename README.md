# Networking Portfolio

This repository contains hands-on networking and troubleshooting labs built using Cisco Packet Tracer.

The goal of this portfolio is to demonstrate practical networking, troubleshooting, and security skills relevant to entry-level IT and networking roles.

---

## 📁 Projects

### 1. VLAN Troubleshooting Lab
- VLAN configuration and segmentation
- Inter-VLAN routing (router-on-a-stick)
- ACL-based security controls
- Real-world troubleshooting scenario

Folder: `/projects/vlan-troubleshooting-lab`

## 🛠 Skills Demonstrated
- Network troubleshooting
